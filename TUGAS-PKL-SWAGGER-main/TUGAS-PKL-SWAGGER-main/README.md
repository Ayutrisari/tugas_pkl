# Project Backend PKL Polinela 2026

Repositori ini berisi kode backend untuk tugas PKL.

## Persiapan Instalasi
1. Clone repositori ini.
2. Jalankan `npm install` untuk mengunduh library.
3. Duplikat file `.env.example` menjadi `.env` dan sesuaikan konfigurasi database Anda.

## Cara Menjalankan Aplikasi
Ketik perintah berikut di terminal:
```bash
node index.js
